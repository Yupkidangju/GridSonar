Summary: GridSonar does not collect, store, or transmit any user data. All processing happens 100% within your browser.

GridSonar ("the App") is a serverless, pure client-side web application. We handle information as follows:

Data Collection

  • GridSonar does not collect any personal information.
  • We do not request or store identifying information such as your name, email, or location.
  • Since there is no server, no data is transmitted externally.

Google Drive Integration

  • When connecting to Google Drive, we only request the drive.file scope.
  • This scope only accesses files that you explicitly select via the Picker UI.
  • We do not scan or browse your entire Drive.
  • Downloaded file data is processed only within your browser memory and never sent to any server.
  • OAuth access tokens are discarded from memory immediately after file download completes.

Local Storage

  • App settings (language, theme, etc.) are stored only in your device's local storage (localStorage, IndexedDB).
  • This data is never transmitted or shared with third parties.
